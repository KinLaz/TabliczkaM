# Tabliczka Mistrza

`Tabliczka Mistrza` to nowoczesna, mobilna aplikacja edukacyjna do nauki tabliczki mnożenia i dzielenia dla dzieci. Aplikacja działa w czystym HTML, CSS i JavaScript, bez backendu i bez żadnych buildów.

## Funkcje

- 4 zakładki: nauka mnożenia, nauka dzielenia, quiz mnożenia, quiz dzielenia
- tryb „Pokaż tabliczkę” i „Ćwicz” dla nauki
- dynamiczne quizy z wyborem ilości pytań
- automatyczna walidacja odpowiedzi
- podsumowanie quizu z punktami, procentem i komunikatem
- zapis zakresu i najlepszych wyników w `localStorage`
- animacje, responsywna obsługa na telefonie i tablecie

## Jak uruchomić lokalnie

1. Skopiuj pliki `index.html`, `style.css`, `script.js` i `README.md` do jednego folderu.
2. Otwórz `index.html` w przeglądarce.

Nie jest potrzebny żaden serwer ani instalacja.

## Jak wrzucić na GitHub

1. Utwórz nowe repozytorium na GitHub.
2. Wypchnij pliki do repozytorium:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
3. Sprawdź, czy struktura jest w katalogu głównym repo.

## Jak włączyć GitHub Pages

1. Wejdź do ustawień repozytorium na GitHub.
2. Przejdź do sekcji `Pages`.
3. Wybierz branch `main` (lub `master`) i folder `/ (root)`.
4. Zapisz ustawienia.

Po chwili aplikacja będzie dostępna pod adresem:

`https://twoj-login.github.io/nazwa-repozytorium/`

## Uwagi

- Aplikacja jest statyczna i działa po prostu po otwarciu `index.html`.
- Używaj tylko względnych ścieżek: `./style.css`, `./script.js`.
